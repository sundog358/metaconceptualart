Ten Nodes

This graph is deliberately small. Its purpose is to establish a repeatable grammar for future works: stable IDs, node types, and relations that can be expanded into a fuller Meta-Wiki.

Node register

Stable IDs, Linked Open Data

Each node has a stable site ID. Where a node names a real concept, movement, or figure, it is grounded in a Wikidata identifier and, for art-domain nodes, the Getty vocabularies — ULAN for people, AAT for concepts — so the museum connects outward to both the general and the art-specific linked-data graph.
